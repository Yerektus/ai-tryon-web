# Dresscode Landing

Маркетинговый лендинг проекта Dresscode на Next.js.

## Что здесь осталось

- лендинг (`/`)
- UI-компоненты лендинга
- базовый layout и стили

Прикладная логика (auth, try-on, billing, social, profile и т.д.) вынесена в соседний проект `dresscode-app`.

## Запуск

```bash
npm install
npm run dev
```

Локальный адрес: [http://localhost:3000](http://localhost:3000)
